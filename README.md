# The Refund Whisper

> People ask for money back long before they say it out loud. I listened to the words in support chats and caught the whisper early.

The question: Can we hear a refund coming before the customer ever asks for one?
The result: Flagged refund intent about two days before it actually happened.

## Run
    pip install -r requirements.txt
    python main.py

## Stack
Python, spaCy, SQL (this demo uses a TF IDF classifier so it runs anywhere).
